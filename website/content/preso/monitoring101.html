Timeseries Graphs --- # Line Graphs ![](../../images/monitoring101/linegraphs.png) Note: Use Line Graphs for the same metric reported by diff scopes to spot outliers at a glance for tracking single metrics to see evolution over time related metrics with same units to see correlations metrics with clear acceptable domain --- # Stacked Area Graphs ![](../../images/monitoring101/stackedareagraphs.png) Note: Same metric from different scopes, to check both sum and contribution of each sum complementary metrics sharing the same unit If the metrics can't be added sensibly, don't use this --- # Bar Graphs ![](../../images/monitoring101/bargraphs.png) Note: use this for sparse metrics to convey values without jumpy or misleading interpolation metrics that represent a count and not a gauge --- # Heat Maps ![](../../images/monitoring101/heatmap.png) Note: single metrics reported by a large number of groups to see trends and variations across a group But if there are few sources, not so useful --- # Metric Graphs 101 ##
